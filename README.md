## Usage
Put "amber.tmLanguage" file in "User" Directory

- Windows: `%APPDATA%\Sublime Text 3\Packages\User`  
- MacOSX: `~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User`  

and restart Sublime Text.
